# Game Overview

## Core Concept
A proper top-down roguelike that innovates on traditional combat by implementing a Wizardry-style perspective for regular encounters, while maintaining special Soulslike elements for boss battles. The game focuses on single-player experience with multiple class options, including a necromancer class.

## Key Features

### Combat Innovation
- Unique blend of classic roguelike and Wizardry-style combat
- Special Soulslike boss encounters
- Multi-stage combat system
- Complex stealth and perception mechanics

### Character Development
- Multiple and natural character builds
- Extensive perk system
- Stat and modifier based progression
- Experience-based perk unlocking

### Survival Elements
- Hunger and thirst management
- Resource quality consideration
- Risk/reward consumption choices

### Visual and Audio Design
- ASCII art with modern touches
- Retro-inspired sound design
- Retro gaming influences

## Design Philosophy
The game aims to blend classic roguelike elements with modern gaming concepts from JRPs and others.
