# Graphics

## Visual Style

### Display Settings
- Resolution: 16:9 at 480p borderless
- Scaling options:
  - Nearest neighbor
  - Sharp bilinear scaling
  - Custom LCD filter option

### ASCII Art Implementation
- Main game interface uses ASCII art
- Code Page 437 inspired graphics
- Special battle graphics:
  - Outlined dithered enemies in black and white
  - Depth-based darkness (enemies appear darker when further away)
  - Background remains ASCII
- Shops and general UI in ASCII with color
- Sprite sheet implementation for better scaling

### Interface Elements
- Menus use simple ASCII design
- Color implementation for UI elements
- Screen effects:
  - Screen CRT Effect
  - Custom scaling filters
